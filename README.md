# class-search [![Build Status](https://travis-ci.org/kudokk/class-search.svg?branch=master)](https://travis-ci.org/cognitom/class-search)

serach your no-use-classes.


## setting
set your search directory in 'search'.
```Gruntfile.js

grunt.initConfig({
  search: {
    dir: 'dist'
  }
});
```

## example directory
```
dist <-search.dir
 |-css
 | |_index.css
 |
 |-html
 | |_index.html
```
